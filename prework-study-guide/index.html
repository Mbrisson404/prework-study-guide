HTML

  • HTML is the structure of the webpage
  • Elements - an element is a type of content on a webpage. Types of elements: text elements, image elements, header elements, list elements.
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Metadata, which is held within the head element, is information about the webpage that is not shown to the user.
  • Tags - the opening tag <> is used to tell the browser to start a process, the ending tag is used to tell the browser to end the process.
  • An HTML attribute defines additional information about an element, eg. lang="eng". HTML attributes are always defined in the beginning element, never the closing element.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Adding colours can be done by using semantic references - like grey or blue - or by using the colours hexadecimal code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd (directory name): navigates to the needed directory.
  • code . : opens VS code
  • git add -A: The add command adds mods in the current working branch and staging area. The -A flag indicates that we want to add all changes.
  • git commit -m "message": commits the files and adds message that describes what the commit contains.
  • git pull origin main: pulls down any changes from the base branch.
  • git push origin branch-name: the push command is used to push the changes we have locally to our remote GitHub branch.

JavaScript

  • JavaScript is used to add features to the webpage and make them interactive.
  • Variables: can be thought of as named containers. The variable needs to be declared before it can be used by using the var keyword.
  • console.log allows us to print the code into the console
  • Conditional/if statements allows JavaScript to make decisions based on a set of conditions. When using an if statement the computer will check to see if the condition is truthy (Anything that is falsy can be seen as a negative input, ie. 0, empty strings, or null). If the statement is truthy it will execute, however, if it is falsy the computer will move on to the else statements until it finds one that is truthy.
  • === checks to see if the two values are equal.
  • Arrays: a single variable that is used to hold a group of data or a list. Made by declaring a variable and then adding the data into square brackets [].
  • For loop: tells the computer to go through each of the items in an array and log them to the console until there are no items left.
  • Functions: a set of instructions that tells the computer how to perform a certain task. You must call a function by name in the console after is declared in order to execute it. A functions name is followed by parentheses, eg. function helloWorld()
  • Control flow is the order in which a computer executes code in a script.
  • Math.floor() and Math.random() are built-in features of JavaScript that can be used together to generate a random number.
